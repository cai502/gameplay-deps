## gameplay-deps v4.0.0 (super)

Packages dependencies for gameplay framework. 

## Building packages:

Windows
-------
- Install Visual Studio 2019
- Install CMake
- Install Python
- Run build.bat

Linux / MacOS
-------------
- Run ./setup.sh
- Run ./build.sh
